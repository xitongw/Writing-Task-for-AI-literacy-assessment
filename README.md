# 离线写作研究平台
本项目是一款面向二语（L2）学术写作研究场景的**离线运行**研究数据采集平台，所有交互记录与文件打包均在浏览器本地完成，无需依赖后端服务，保障数据隐私与离线可用性。

## 核心功能模块

### 1. AI素养评估（AI Literacy Assessment）
- 单选题形式评估受试者AI素养相关能力
- 支持进度保存、未完成题目标记功能

### 2. 问卷调研（Questionnaire）
- 包含背景信息采集 + 6点量表题（含可选补充条目）
- 支持进度保存、未完成题目标记功能

### 3. 写作任务（Writing Task）
基于 TipTap 富文本编辑器构建，**精细化记录所有编辑行为轨迹**，完整捕捉写作过程数据：

#### A. 基础编辑行为记录
##### （1）文本输入（Insert）
- **判定逻辑**：监听编辑器`onTransaction`事件，检测文本节点新增内容（排除粘贴行为）
- **缓冲机制**：实现2秒输入缓冲（`insertBuffer`），避免逐字记录：
  - 连续快速输入会被合并
  - 仅在输入停顿超2秒、编辑器失焦或触发其他操作（如删除）时，统一记录为完整的`insert`事件
- **记录内容**：输入文本、时间戳、光标位置

##### （2）文本删除（Delete）
- **判定逻辑**：在`onTransaction`事件中检测以下场景：
  - 选区范围缩小（`to > from`）且无替换内容
  - 替换内容长度小于原选中内容
- **记录规则**：立即记录删除的文本内容（无缓冲）
- **记录内容**：删除文本、时间戳、删除位置

##### （3）粘贴（Paste）
- **判定逻辑**：通过编辑器`handlePaste`属性拦截粘贴事件
- **特殊处理**：直接读取剪贴板`text/plain`数据并记录为`paste`事件，避免被误判为普通输入
- **记录内容**：粘贴文本、时间戳

#### B. 剪贴板操作记录
##### （1）复制（Copy）
- **判定逻辑**：监听`handleKeyDown`事件，检测`Meta/Ctrl + C`快捷键
- **触发条件**：仅当编辑器内有选中文本时记录
- **记录内容**：复制文本、时间戳

##### （2）剪切（Cut）
- **判定逻辑**：监听`handleKeyDown`事件，检测`Meta/Ctrl + X`快捷键
- **触发条件**：仅当编辑器内有选中文本时记录
- **记录内容**：剪切文本、时间戳、剪切后光标位置

#### C. 自定义交互行为
##### （1）标记红字（Mark Red）
- **触发方式**：点击工具栏「Mark Red」按钮
- **行为逻辑**：获取当前选中文本并设置为红色，同时记录事件
- **记录内容**：`mark_red`事件类型、选中文本、时间戳、选区范围

##### （2）标记蓝字（Mark Blue）
- **触发方式**：点击工具栏「Mark Blue」按钮
- **行为逻辑**：获取当前选中文本并设置为蓝色，同时记录事件
- **记录内容**：`mark_blue`事件类型、选中文本、时间戳、选区范围

#### D. 状态监控
##### （1）空闲状态（IDLE_START / IDLE_END）
- **空闲开始（IDLE_START）**：
  - 每0.5秒检测一次用户行为（鼠标移动/点击、键盘输入）
  - 触发条件：无操作时长超2秒 且 窗口处于激活状态
  - 记录内容：`IDLE_START`事件、时间戳
- **空闲结束（IDLE_END）**：
  - 检测到任何用户操作（鼠标/键盘）立即触发
  - 仅当此前处于空闲状态时记录`IDLE_END`事件、时间戳

##### （2）焦点变化（Focus / Blur）
- **失去焦点（Blur）**：监听`window.blur`事件，强制触发输入缓冲结算（`flushBuffer`），确保未完成的输入记录被保存
- **获取焦点（Focus）**：监听`window.focus`事件，记录`focus`事件、时间戳

#### E. 辅助功能
- 实时字数统计（Word count）：内容变化时动态更新
- 一键导出提交包：点击下载生成ZIP文件，包含：
  - 核心文件：写作内容（`content.txt`纯文本 / `content.html`格式化文本）、完整编辑日志（`log.json`）
  - 附加文件：AI素养评估答案（`assessment_answer.json`）、问卷调研结果（`questionnaire_answer.json`）（若已保存进度）

### 4. AI交互数据提取（Extract AI Interaction Data）
- 支持从 Deepseek 导出的 ZIP/JSON 文件中，按标题筛选目标对话
- 生成过滤后的 ZIP 文件，仅保留符合条件的交互数据

## 技术栈
- 前端框架：React 18
- 构建工具：Vite 5
- 富文本编辑：TipTap
- 本地文件处理：JSZip（ZIP打包/解包）
- 构建优化：vite-plugin-singlefile（生产构建优先内联为单文件）
